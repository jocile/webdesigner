  1. Titulo

    A tag h1 é usada para definir o título principal de uma página ou de uma seção específica, sendo o maior e mais importante título em termos de hierarquia.

    A tag h2 é semelhante ao h1, porém é usada para títulos secundários, sendo ligeiramente menores em tamanho e importância.

    A tag h3 é utilizada para títulos de terceiro nível, geralmente empregada para subdividir seções importantes de uma página.

    A tag h4 é empregada para títulos de quarto nível, oferecendo uma hierarquia de título menor que h3.

    A tag h5 é usada para títulos de quinto nível, sendo menores que os títulos anteriores e geralmente empregados para subdividir conteúdo mais específico.

    A tag h6 é a menor em termos de hierarquia de títulos, sendo utilizada para subdividir conteúdo ainda mais específico dentro de uma página.

  2. Paragrafo

    A tag p é usada para criar parágrafos de texto, ajudando a estruturar e organizar o conteúdo de uma página web de maneira clara e legível.

  3. Grifar Texto

    A tag mark é utilizada para destacar parte do texto, geralmente com um fundo colorido, para chamar a atenção do leitor para informações importantes ou termos específicos.

  4. Small

    A tag small é usada para indicar que o texto contido dentro dela é de tamanho reduzido em relação ao texto circundante, geralmente utilizado para notas de rodapé ou informações secundárias.

  5. Ítalico

    A tag i é utilizada para enfatizar o texto, geralmente representando texto em itálico, mas não necessariamente para denotar ênfase.

  6. Sublinhar Texto

    A tag u é usada para sublinhar o texto, geralmente indicando que o texto é uma referência ou um link.

  7. Negritar Texto

    A tag strong é utilizada para destacar o texto de forma mais forte que o texto normal, indicando importância semântica ou ênfase.

  8. Lista ordenada

    A tag ol é usada para criar listas ordenadas, onde os itens são numerados automaticamente.

  9. Lista sem ordem

    A tag ul é usada para criar listas não ordenadas, onde os itens são marcados com pontos ou outros símbolos, mas não são numerados automaticamente.

  10. Itens da lista

    A tag li é utilizada dentro de ol ou ul para definir cada item da lista, seja ordenada ou não.

  11. links

    A tag a é usada para criar links, permitindo que os usuários naveguem entre diferentes páginas da web ou dentro da mesma página.

  12. Linha Divisoria

    A tag hr é utilizada para criar uma linha horizontal, separando o conteúdo da página em seções distintas.

  13. Subscrever texto

    A tag sub é usada para colocar o texto em formato de subscrito, geralmente utilizado para fórmulas químicas ou equações.

  14. Sobreescreve texto

    A tag sup é utilizada para colocar o texto em formato de sobrescrito, como em expoentes ou índices.

  15. Citação

    A tag blockquote é usada para citar blocos de texto longos ou citações de outras fontes, geralmente com um recuo à esquerda para destacar visualmente a citação.
