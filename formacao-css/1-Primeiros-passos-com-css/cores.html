Cores com CSS

Existem várias forma de definir cores utilizando CSS

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL - matriz, saturação e luminosidade
  • HSLA - com transparência
